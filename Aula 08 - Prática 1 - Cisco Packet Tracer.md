## Cisco Packet Tracer

- O **Cisco Packet Tracer** é um software de simulação de redes criado pela **Cisco Networking Academy**, muito usado para ensinar e praticar conceitos de redes de computadores sem precisar de equipamentos físicos.

---

## Para que ele serve?

- Montar redes virtualmente.
- Conecte equipamentos como:
  - Roteadores;
  - Switches;
  - Computadores.
  - Servidores
- Configure protocolos (IP, DHCP, DNS, etc.)
- Simule o tráfego de dados na rede.
- Teste cenários reais (ping, falhas, roteamento).

---

## O que conseguimos aprender com ele?

- Você consegue praticar toda a base de redes:
  - Endereçamento IP
  - Sub-redes (subnetting)
  - Roteamento (estático e dinâmico)
  - VLANs
  - NAT
  - DHCP e DNS
  - Topologias de rede

---

## Principais Modos

- **Modo Tempo Real (Realtime)**
  - A rede funciona como no mundo real.
  
- **Modo Simulação (Simulation)**
  - Você vê os pacotes passando pelos dispositivos.

---

## Primeira Atividade - Computadores Conectados

- Coloque dois computadores na área de trabalho.
- Conecte-os através de um cabo crossover.
- Configure os IPs dos computadores.
- Testes os pacotes trafegando pela rede.
- Execute o comando ping entre os computadores.

---

## Responda sobre Primeira Atividade

- O ping funcionou? O que isso significa?
- Os dois computadores estão na mesma rede? Como você sabe?

## Segunda Atividade - Utilizando o HUB

- Coloque pelo menos 3 computadores em sua área de trabalho.
- Coloque um hub em sua área de trabalho.
- Conecte os computadores ao HUB.
- Configure os IPs
  - PC0 - 192.168.0.2
  - PC1 - 192.168.0.3
  - PC2 - 192.168.0.4

- Testes e observe os pacotes trafegando pela rede.
